# PyWall Architect 🛡️

Generatore di regole firewall con Streamlit e SQLite.

## Installa e Avvia
1. `pip install -r requirements.txt`
2. `streamlit run app.py`

## Funzionalità
- Genera sintassi per iptables, nftables, Cisco ASA
- Salva/elimina regole in SQLite
- Validazione IP automatica
